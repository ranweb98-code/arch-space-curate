

# Architecture Studio Website

A premium, minimalist website for an architecture studio — clean, confident, and timeless.

---

## Design System

- **Colors**: White background, soft grays (#F5F5F0 stone), black accents, minimal color
- **Typography**: Strong hierarchy — large serif headings (elegant, architectural feel), clean sans-serif body text
- **Spacing**: Generous whitespace, grid-based layouts, large imagery
- **Animations**: Subtle fade-in on scroll, smooth transitions between pages

---

## Pages

### 1. Homepage
- **Hero**: Full-width architectural image with subtle dark overlay, bold headline ("Architecture With Vision"), short subtitle, two CTA buttons (View Projects / Contact)
- **Studio snapshot**: Brief intro with philosophy teaser
- **Featured projects**: 3 highlighted projects in a grid
- **Services overview**: Quick summary linking to full services page
- **Process preview**: Visual 4-step flow
- **CTA banner**: Encouraging contact

### 2. About
- Studio introduction with design philosophy
- Focus on attention to detail and client-centered approach
- Supporting imagery alongside clean text layout

### 3. Projects
- Grid-based portfolio page with hover effects (subtle zoom/overlay with project name)
- Each project links to a dedicated detail page with:
  - Image gallery (multiple photos)
  - Project description, location, type, year
  - Navigation to next/previous project
- Uses placeholder architectural images initially (easily swappable)

### 4. Services
- Clean card layout with icons for each service:
  - Residential Architecture
  - Commercial Architecture
  - Interior Architecture
  - Renovations & Planning
- Brief description per service, structured for easy editing

### 5. Pricing
- Three tiers presented as elegant cards:
  - **Concept & Planning** — starting range, what's included
  - **Full Architectural Project** — comprehensive package
  - **Custom Projects** — "Request a Quote" with contact CTA
- Professional tone emphasizing value and clarity

### 6. Process
- Visual step-by-step flow (4 steps):
  1. Initial Consultation
  2. Concept & Design
  3. Development & Planning
  4. Execution & Delivery
- Clean numbered sections with icons and short descriptions

### 7. Contact
- Minimal contact form (name, email, project type, message)
- Studio details (address, phone, email)
- Clear CTA, professional layout

---

## Navigation & Layout
- Fixed top navigation bar — logo left, page links right, "Contact" as a subtle button
- Smooth scroll behavior throughout
- Mobile-responsive hamburger menu
- Consistent footer with studio info, quick links, and social icons

---

## Technical Notes
- All content is hardcoded (no backend needed) — easy to swap text and images
- Fully responsive, mobile-first design
- Scroll-triggered fade-in animations for sections
- Placeholder images from Unsplash (architecture photos)
- SEO-friendly semantic HTML structure
